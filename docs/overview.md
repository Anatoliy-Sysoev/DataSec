# Обзор решения

DataSec Reference Architecture — это обезличенное референсное решение для построения защищенной аналитической платформы.

## Назначение

Решение описывает, как организовать безопасный доступ к данным через Trino с использованием:

- Keycloak для SSO-аутентификации;
- LDAP / FreeIPA / AD для групп доступа;
- Apache Ranger для авторизации и аудита;
- IDM-процесса для заявок и жизненного цикла доступов;
- OpenSearch / Solr / log pipeline для передачи audit-событий в SIEM.

## Основная идея

Аутентификация и авторизация разделяются:

- **Keycloak** отвечает на вопрос: «Кто этот пользователь?»
- **Apache Ranger** отвечает на вопрос: «Что этому пользователю разрешено делать с данными?»
- **LDAP-группы** отвечают на вопрос: «В какой роли доступа состоит пользователь?»
- **IDM** отвечает на вопрос: «Кто согласовал доступ и когда его нужно отозвать?»

## Для кого репозиторий

- системных аналитиков;
- Data Platform инженеров;
- DevOps / SRE;
- специалистов ИБ;
- архитекторов платформ данных;
- команд, внедряющих Trino и Ranger.
